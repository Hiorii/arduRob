# ArduRob ecommerce app

## Live demo

https://ardu-rob.herokuapp.com/

## Api

https://ardu-rob.herokuapp.com/api

## How to use

#### Endpoints:

`api/products` 

#### To local run try:

backend server start:

`cd server && yarn start`

frontend start:

`cd client && yarn start`

### Features and stack

- @reactJS
- @react-hooks
- @redux
- @redux-thunk
- @expressJS
- @MongoDB
- @mongoose
